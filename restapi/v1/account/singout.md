# Выход из аккаунта
> POST /v1/account/singout

## Запрос

Параметр | Обязательный | Описание
-|-|-
token | Да | Токен

## Ответ

### 200 OK
> Выход успешно проиведён

### 400 Bad Request
> Не указан обязательный параметр

Поле | Тип | Описание
-|-|-
error | STRING | Сообщение об ошибке

```json
{
    'error': STRING
}
```

### 401 Unauthorized
> Указан недействительный токен

Поле | Тип | Описание
-|-|-
error | STRING | Сообщение об ошибке

```json
{
    'error': STRING
}
```
